# 🚀 HTTPS 快速测试 - 3 步解决

## 第 1 步：访问 HTTPS 地址

用手机浏览器打开：

```
https://172.20.10.2:5173
```

## 第 2 步：跳过安全警告

看到安全警告时：

- **Android**: 点击"高级" → "继续前往"
- **iOS**: 点击"显示详细信息" → "访问此网站"

## 第 3 步：测试媒体功能

1. 进入**语音分析**页面
2. 点击"开始声音收集"
3. 允许麦克风权限
4. 应该能看到实时声波图！

## 🎯 成功标志

- ✅ 页面正常加载
- ✅ 没有"浏览器不支持媒体设备访问"错误
- ✅ 能看到实时声波动画
- ✅ 麦克风权限正常工作

## 🔧 如果还有问题

检查服务器是否在运行，或尝试其他 IP 地址：

- `https://localhost:5173`
- `https://192.168.1.xxx:5173`
